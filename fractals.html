Koch Curve

Default curve: koch_curve(t, 6, 50)

This Koch curve has a number of modifications:

  • The color of the Koch curve is now 'steelblue.'
  • The Koch curve forms a distinct pattern as it now turns a random multiple of 60 degrees right at the end of a cycle.

y = random.randrange(20)
def koch_curve(t, depth, length):
    if depth == 1:
        t.color('steelblue')
        t.fd(length)
    else:
        koch_curve(t, depth - 1, length)
        t.lt(60)
        koch_curve(t, depth - 1, length)
        t.rt(120)
        koch_curve(t, depth - 1, length)
        t.lt(60)
        koch_curve(t, depth - 1, length)
        t.rt(60 * y)

Sierpinski Triangle

Default triangle: sierpinski(t, 5, 50)

This Sierpinski Triangle has a number of modifications:

  • Instead of going forwards, the turtle draws triangles outlining the original triangles instead of in the original triangle.
  • The Sierpinski Triangle no longer draws the first leg of the triangle.

def sierpinski(t, depth, size, scale_factor=1):

        t.lt(120)

Trees

Default tree: tree(t, 4, 50, 20)

This tree has a number of modifications:

  • Every time the turtle retraces its steps, it goes backwards by an additional constant random number.
  • The "leaves" are green while the branches are given different shades depending on their strokes. The first stroke is saddlebrown, the second is brown, the third is sandybrown, and the last is chocolate.
tree(t, 4, 50, 20) 

        
def tree(t, depth, size, angle):
    x = random.randrange(20)
    if depth == 0:
        t.pencolor("green")
        t.fd(size)
        t.bk(size + x)
    else:
        t.pencolor("saddlebrown")
        t.fd(size)
        t.rt(angle)
        t.pencolor("brown")
        tree(t, depth-1, size, angle)
        t.pencolor("sandybrown")
        t.lt(2 * angle)
        tree(t, depth-1, size, angle)
        t.pencolor("chocolate")
        t.rt(angle)
        t.bk(size + x)

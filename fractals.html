Koch Curve

Default curve: koch_curve(t, 4, 20)

This is the basic Koch Curve except as the depth increases the size decreases. koch_curve(t, 6, 100) 

        
def koch_curve(t, depth, size):
    if (depth == 1):
        t.fd(size)
    else:
        koch_curve(t, depth-1, size*0.6)
        t.lt(60)
        koch_curve(t, depth-1, size*0.6)
        t.rt(120)
        koch_curve(t, depth-1, size*0.6)
        t.lt(60)
        koch_curve(t, depth-1, size*0.6)

Sierpinski Triangle

Defauly triangle: sierpinski(t, 5, 200)

This function does not actually draw the triangle at the base case, instead what we see is the result of the turtle moving to position itself correctly.sierpinski(t, 6, 100) 

        
def sierpinski(t, depth, size, scale_factor=1):
    if depth > 1:
        sierpinski(t, depth-1, size/2)
        t.fd(size/2)
        sierpinski(t, depth-1, size/2)
        t.bk(size/2)
        t.lt(60)
        t.fd(size/2)
        t.rt(60)
        sierpinski(t, depth-1, size/2)
        t.rt(120)
        t.fd(size/2)
        t.lt(120)

Trees

Default tree: tree(t, 4, 50, 20)

This tree has a number of modifications:

  • Every time the turtle retraces its steps, it goes backwards by an additional constant random number.
  • The "leaves" are green while the branches are given different shades depending on their strokes. The first stroke is saddlebrown, the second is brown, the third is sandybrown, and the last is chocolate.
tree(t, 4, 50, 20) 

        
def tree(t, depth, size, angle):
    x = random.randrange(20)
    y = depth
    if depth == 0:
        t.pencolor("green")
        t.fd(size)
        t.bk(size + x)
    else:
        t.pencolor("saddlebrown")
        t.fd(size)
        t.rt(angle)
        t.pencolor("brown")
        tree(t, depth-1, size, angle)
        t.pencolor("sandybrown")
        t.lt(2 * angle)
        tree(t, depth-1, size, angle)
        t.pencolor("chocolate")
        t.rt(angle)
        t.bk(size + x)
